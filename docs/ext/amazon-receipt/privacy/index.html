プライバシーポリシー

Chrome拡張機能「アマゾン領収書・電帳法索引簿メーカー」／最終更新: 2026年7月12日

要点
  • ご注文の内容・金額・氏名・住所などのデータを外部に送信することは一切ありません
  • すべての処理はお使いのブラウザ内で完結します。当方のサーバーに保存されるデータはありません
  • 作成されたCSV・領収書ファイルはお使いのパソコンにのみ保存されます

1. 取得しないデータ

本拡張機能は、以下のデータを取得・送信・保存しません。

  • Amazonのアカウント情報（メールアドレス・パスワード・Cookie）
  • 注文内容、商品名、金額、注文番号、氏名、住所、支払い方法
  • 閲覧履歴、その他のウェブサイトの情報

注文履歴ページから読み取った情報は、ブラウザのメモリ上でCSVを組み立てるためだけに使われ、処理後に破棄されます。当方のサーバーへ送信されることはありません。

2. 拡張機能が行う通信

本拡張機能が外部と通信するのは、次の2点のみです。

通信先目的送信される内容
keiri-tools.com 抽出ルール（セレクタ定義）の取得。Amazonのページ構造が変わった際に、拡張機能を更新せずに修正できるようにするため なし（定義ファイルを取得するだけで、こちらから情報は送りません）
amazon.co.jp 領収書ページの読み取り（注文履歴に金額が表示されない注文の補完） お使いのブラウザが通常アクセスするのと同じ内容のみ。当方は関与しません

Pro版をご利用の場合、ライセンス確認のために決済事業者（ExtensionPay / Stripe）と通信します。この際に送信されるのはライセンス確認用の識別子とメールアドレスのみで、注文データは一切含まれません。

3. 権限を必要とする理由

権限必要な理由
storage抽出ルールのキャッシュと設定の保存（ローカルのみ）
downloads索引簿CSV・領収書ファイルをお使いのパソコンに保存するため
amazon.co.jp へのアクセス注文履歴ページの読み取りと、領収書ページの取得のため

4. 第三者への提供

取得するデータがないため、第三者へ提供するデータもありません。分析ツール・広告トラッカーの類は一切組み込んでいません。

5. お問い合わせ

本ポリシーに関するお問い合わせは、GitHubのIssueまでお願いします。

6. 改定

本ポリシーを改定する場合は、このページで告知します。
