① コピーする

各ツールの「コピー」ボタンでこの形のコードが取れます。src のツール名を変えるだけで、どの計算機でも使えます。

<iframe src="https://keiri-tools.com/embed/tedori/"
  title="手取り計算" loading="lazy"
  style="width:100%;max-width:480px;height:640px;
  border:1px solid #e2e8f0;border-radius:12px"></iframe>

<script>
addEventListener("message",function(e){
  if(e.data&&e.data.type==="keiri-embed"){
    document.querySelectorAll(
      'iframe[src*="keiri-tools.com/embed/"]'
    ).forEach(function(f){
      if(f.contentWindow===e.source)
        f.style.height=e.data.height+"px";
    });
  }
});
</script>

※ 2つ目の <script> は中身の高さに合わせて枠を自動調整します（省略時は固定640px）。複数のウィジェットを貼っても1つで足ります。

使い方

  1. 使いたいツールの「コピー」を押す
  2. WordPress なら「カスタムHTML」ブロック、その他は記事の HTML に貼る
  3. 公開して完了。以後こちらで料率・税額表を最新に保つので、貼りっぱなしで大丈夫です
