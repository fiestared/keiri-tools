消費税・端数処理 計算機
税込・税抜を8%/10%で相互計算。インボイス（適格請求書）の消費税額を「税率ごとに1回」の端数処理で算出し、認められない明細ごとの端数処理との差額も表示。
社会保険料 計算機
給与額から標準報酬月額の等級・健康保険・介護保険・厚生年金を自動計算。令和8年度の都道府県別料率対応。賞与も上限つきで計算。
源泉徴収税額 計算機
給与は国税庁の税額表（月額表・令和8年分／甲欄・乙欄）、賞与は算出率の表を引いて計算。扶養親族等の数にも対応。報酬・原稿料・士業への支払い（10.21%）も計算できます。
失業保険（基本手当）計算機
離職時の年齢・給料・勤続年数・退職理由から、1日あたりの基本手当日額・所定給付日数・受け取れる総額の目安・支給が始まる時期を計算。会社都合と自己都合の違い、給付制限にも対応。
有給休暇の付与日数 計算機
入社日から付与日数・付与日を自動計算。パートの比例付与、年5日の取得義務の判定にも対応。
日数計算・営業日計算
期間の日数、「◯営業日後」の日付、期間内の営業日数を計算。土日・祝日・年末年始に対応。
Amazon領収書 → 電帳法の索引簿Chrome拡張
Amazonの注文履歴から索引簿CSVを1クリックで作成。キャンセル注文は自動で除外します。
電帳法 索引簿テンプレート（無料DL）
電子帳簿保存法の検索要件を満たす索引簿のCSVテンプレート。記入例つき・書き方の解説つき。
支払サイト・支払日計算
「月末締め翌月末払い」等から向こう1年の支払日を一覧化。土日祝は営業日調整、カレンダー出力も。

高額療養費の自己負担限度額【令和8年】区分は年収でなく標準報酬月額で決まる
限度額の区分を決めるのは年収ではなく標準報酬月額。報酬月額が1円違うだけで等級が上がり、医療費100万円のときの自己負担が87,430円から171,820円に変わる。世帯合算は21,000円未満を拾わない。マイナ保険証を使う人は、そもそも限度額適用認定証を申請できない（省令が交付対象を資格確認書を持つ人に限っている）。

育児休業給付金はいくら？67%＋13%の計算と「手取り10割」の条件
育児休業給付金は賃金日額の67%（181日目以降50%）。出生後休業支援給付金13%を足すと80%＝手取り10割相当だが、母親側はこの13%に「配偶者が14日以上育休を取る」という条件が付く。父親は妻の産後休業で自動的に要件を満たすため、この不利は母親だけに生じる。

手取りはいくら？額面から手取りを計算する方法【令和8年度】
手取りは額面の約8割。だが社会保険料は等級表の階段で決まるので、額面309,999円の人より310,000円の人のほうが手取りが2,837円少ない。所得税は同じ行に収まって1円も動かないため、社会保険料の増加がまるごと手取りの減少になる。住民税は去年の所得にかかるので、新卒1年目には引かれない。

失業保険はいくらもらえる？計算方法と自己都合・会社都合の違い
失業保険の日額は離職理由では1円も変わらない。変わるのは所定給付日数と開始時期で、35歳・月30万・勤続12年なら自己都合744,840円・会社都合1,489,680円と総額が倍になる。34歳と35歳の間には30日（186,210円）の崖があり、自己都合で勤続1年未満なら1円も出ない。

再就職手当はいくらもらえる？計算方法と支給条件【令和7年度】
再就職手当＝基本手当日額×支給残日数×60%（残日数が所定給付日数の3分の2以上なら70%）。日額には6,570円の上限があり高給の人ほど削られる。所定120日なら残80日と79日の境目で53,381円の崖があり、1日早く決めるほうが得。就業手当は2025年4月に廃止。
