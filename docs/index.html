消費税・端数処理 計算機
税込・税抜を8%/10%で相互計算。インボイス（適格請求書）の消費税額を「税率ごとに1回」の端数処理で算出し、認められない明細ごとの端数処理との差額も表示。
社会保険料 計算機
給与額から標準報酬月額の等級・健康保険・介護保険・厚生年金を自動計算。令和8年度の都道府県別料率対応。賞与も上限つきで計算。
源泉徴収税額 計算機
給与は国税庁の税額表（月額表・令和8年分／甲欄・乙欄）、賞与は算出率の表を引いて計算。扶養親族等の数にも対応。報酬・原稿料・士業への支払い（10.21%）も計算できます。
有給休暇の付与日数 計算機
入社日から付与日数・付与日を自動計算。パートの比例付与、年5日の取得義務の判定にも対応。
日数計算・営業日計算
期間の日数、「◯営業日後」の日付、期間内の営業日数を計算。土日・祝日・年末年始に対応。
電帳法 索引簿テンプレート（無料DL）
電子帳簿保存法の検索要件を満たす索引簿のCSVテンプレート。記入例つき・書き方の解説つき。
支払サイト・支払日計算
「月末締め翌月末払い」等から向こう1年の支払日を一覧化。土日祝は営業日調整、カレンダー出力も。
