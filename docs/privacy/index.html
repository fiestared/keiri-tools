プライバシーポリシー

最終更新: 2026年7月13日

要点
  • ツールに入力した金額・日付・名義などのデータは、サーバーに送信されません。すべてお使いのブラウザ内で処理され、当サイトが受け取ることはありません
  • 広告配信（Google AdSense）とアクセス解析（Google Analytics）のために、Cookieを利用しています
  • Cookieによる広告の最適化は、ご自身で無効にできます（下記に手順あり）

1. ツールに入力されたデータの扱い

当サイトの計算ツール（社会保険料計算機、源泉徴収税額計算機、振込手数料計算、振込名義カナ変換、日数計算、支払サイト計算、有給休暇計算など）は、すべてお使いのブラウザ内（JavaScript）で計算しています。

入力された給与額・報酬額・取引先名・日付などのデータが、当サイトのサーバーや第三者へ送信されることは一切ありません。当方がこれらのデータを取得・保存・閲覧することもできません。

一部のツールでは、入力内容をブラウザのローカルストレージに保存する機能があります（支払サイト計算ツールの取引先条件など）。この保存先もお使いの端末内のみで、外部には送信されません。ブラウザのデータを消去すれば削除されます。

2. 広告配信について（Google AdSense）

当サイトは、第三者配信の広告サービス「Google AdSense」を利用しています。

Googleなどの第三者配信事業者は、Cookieを使用して、ユーザーが当サイトや他のサイトに過去にアクセスした際の情報に基づいて広告を配信することがあります。

Googleが広告Cookieを使用することにより、ユーザーは広告設定でパーソナライズ広告を無効にできます。また、aboutads.info で第三者配信事業者のCookieを無効にすることもできます。

詳しくは、Googleの広告に関するポリシーをご確認ください。

3. アクセス解析について（Google Analytics）

当サイトでは、サイトの改善のためにアクセス状況を把握する目的で、Googleが提供するアクセス解析ツール「Google Analytics 4」を利用しています。

Google Analyticsは、トラフィックデータの収集のためにCookieを使用します。収集されるのは、閲覧されたページ・訪問時刻・参照元・おおまかな地域・使用しているブラウザや端末の種類といった情報で、氏名・メールアドレスなど個人を特定する情報は含まれません。

ツールに入力された金額・取引先名などのデータが送信されることはありません（それらはブラウザ内でのみ処理されます）。

Google Analyticsによるデータ収集を望まない場合は、Google アナリティクス オプトアウト アドオンを導入するか、ブラウザの設定でCookieを無効にすることで拒否できます。

詳しくはGoogleのポリシーと規約をご確認ください。

4. Chrome拡張機能について

当サイトが公開しているChrome拡張機能「アマゾン領収書・電帳法索引簿メーカー」については、拡張機能のプライバシーポリシーに別途記載しています。注文データを外部に送信することは一切ありません。

当サイトのコンテンツの著作権は、当方または引用元に帰属します。引用の範囲を超えて無断で転載・複製することはご遠慮ください。ツールへのリンクは自由に行っていただけます。
